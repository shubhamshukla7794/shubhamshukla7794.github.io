My Recipe Application

The project My Recipes App a spring sample application developed while undergoing an online course called Spring Framework 5: Beginner to Guru.

In this application, the user can view all the recipes on the Main Page. Here, the user can also explore the recipes based on the categories available. In this application, you can add a 'New Recipe' or you can update an existing recipe. One can also delete an existing recipe. There is also a landing page for '404 Error' as well as a 'Bad Request' error.

Running Application Process on your computer

My Recipes App is a Spring Boot application built using Maven.

  • Download the application from the above

  • Unzip the application

  • Open your terminal

  • Move to the root of the application

  • Run mvn spring-boot:run

  • Navigate to http://localhost:8080 in your default browser

Working with Recipe App in your IDE

Prerequisites

The following items should be installed in your system:

  • Java 11 or newer

  • Download and Install Maven

  • Your preferred IDE

Database Configuration

In its default configuration, Recipe App uses an in-memory database (H2) which gets populated at startup with data. The H2 console is automatically exposed at http://localhost:8080/h2-console and it is possible to inspect the content of the database using the jdbc:h2:mem:testdb url.
