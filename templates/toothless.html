Toothless Pet Clinic Application

The project Toothless Pet Clinic is my version of the classic spring project "Spring Pet Clinic". This application was developed while undergoing an online course called Spring Framework 5: Beginner to Guru.

The application consists of the interaction with a pet clinic, by creating owners, pets, visits, etc. The user can view the list of all the pet owners. The user can also search for a particular pet owner from the Last Name of the owner. They can add details of the pet associated with the owner as well as schedule the visit to the Vet.

This application was built using Java 11, Spring 5, Spring Boot 2, Hibernate ORM, H2 Database with Thymeleaf for rendering the UI.


Running Application Process on your computer

Toothless Pet Clinic is a Spring Boot application built using Maven.

  • Download the application from the above

  • Unzip the application

  • Open your terminal

  • Move to the root of the application

  • Run mvn spring-boot:run

  • Navigate to http://localhost:8080 in your default browser


Working with Recipe App in your IDE

Prerequisites

The following items should be installed in your system:

  • Java 11 or newer

  • Download and Install Maven

  • Your preferred IDE


Database Configuration

In its default configuration, Toothless Pet Clinic uses an in-memory database (H2) which gets populated at startup with data. The H2 console is automatically exposed at http://localhost:8080/h2-console and it is possible to inspect the content of the database using the jdbc:h2:mem:testdb URL.
